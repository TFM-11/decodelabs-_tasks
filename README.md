Number Guessing Game

Overview
The Number Guessing Game is a Java console application developed as part of the DecodeLabs Java Programming Industrial Training Kit (Project 1).

The application generates a random number within a specified range and challenges the user to guess it. The program provides hints after every attempt until the correct number is guessed.

This project focuses on strengthening fundamental Java programming concepts such as loops, conditional statements, user input, and random number generation.

Features

- Generate a random number between 1 and 100
- Accept user guesses through the console
- Inform the user whether the guess is:
  - Too High
  - Too Low
  - Correct
- Keep track of the number of attempts
- Display a success message after guessing correctly
- Input validation to prevent invalid entries


Technologies Used

- Java
- Scanner Class
- Random Class
- IntelliJ IDEA / VS Code
- JDK 17+
